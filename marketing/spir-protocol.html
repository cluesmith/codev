The SPIR Protocol

Specify
📋
  • Builder drafts spec (the what)
  • 3-way review (Codex, Gemini, Claude)
  • Builder addresses or rebuts feedback
  • Spec block on final human review + approval
Plan
🗺️
  • Builder drafts plan (the how)
  • Breaks down project into phases
  • 3-way review
  • Builder addresses or rebuts feedback
  • Plan block on final human review + approval
Implement
🔨
  • For each phase:
  • Builder implements phase
  • 3-way review + run test suite
  • Repeat until tests and reviews are good
Review
🔍
  • Create a PR
  • Final 3-way review
  • Verify product meets criteria in spec
  • Write review
  • Update architecture / lessons learned
  • Tell architect it's done
